- Python3
- Visual Studio for C++ <!-- (If on windows. Another Linux win.) -->

## Installation
- `git clone https://github.com/TardisKing40/python-facerec.git`
- `cd python-facerec`
- `pip install -r requirements.txt`
- `python3 facerec.py`
